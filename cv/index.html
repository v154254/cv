E-Commerce App

Приложение написано на базе предоставленной сайтом commercetools.com.

Сайт представляет из себя образец электронного магазина. Commercetools берёт на себя работу с бэкэндом — позволяет настроить скоупы (разрешения) для пользователей, указать товары для продажи и цены на них.

Данное приложение:

  • Является single-page application
  • Обладает бесшовным роутингом между страницами
  • Принимает и корректно обрабатывает токены аутентификации
  • Принимает и отправляет данные с использованием API commercetools

С точки зрения пользователя, в приложении можно:

  • Зарегистрироваться
  • Посмотреть указанные при регистрации данные и изменить их
  • Ознакомиться со списком товаром
  • Произвести сортировку и/или фильтрацию товаров
  • Произвести поиск товара по названию
  • Добавить или удалить товары из корзины

Momentum

Приложение вдохновлено одноимённым приложением которое можно найти в Google Store.

Приложение позволяет пользователю указать своё имя, выбрать город для просмотра погоды и указать язык в настройках. Все указанные данные будут сохранены в LocalStorage. Также приложение показывает время и дату в реальном времени, позволяет слушать музыку и менять фон в зависимости от времени суток или выбранных пользователем тэгов.

Приложение является технически несложным и демонстрирует, что даже на нативном JS можно написать приятно выглядящее приложение с относительном большим числом разнообразных функций.

Виртуальная клавиатура

Приложение представляет из себя виртуальную клавиатуру.

В данной клавиатуре можно менять язык раскладки, включать Caps Lock и зажимать Shift для смены символов. Все клавиши работают как при нажатии на реальной клавиатуре, так и при нажатии мышью по клавишам виртуальной клавиатуры.

Приложение демонстрирует навыки обращения с KeyboardEvent.
